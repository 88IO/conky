# Conky Config
For UltraWide Monitor 2560x1080

## Required
- conky with lua and nvidia enabled
```
sudo pacman -S conky-lua-nv
```
OR
```
yay -S conky-cairo
```

## Setup
```
git clone https://github.com/88IO/conky-conf ~/.config/conky
```
